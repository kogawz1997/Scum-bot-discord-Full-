Step by Step

เริ่มจากพื้นที่ทดลอง แล้วค่อยเปิด runtime ที่จำเป็น

ไม่ต้องเปิดทุกระบบพร้อมกันตั้งแต่วันแรก เริ่มดูเมนู สิทธิ์ และฟีเจอร์ก่อน แล้วค่อยตัดสินใจเปิด Delivery Agent หรือ Server Bot ตามงานจริง

สร้างบัญชี เปิดพื้นที่ทดลอง
1

สร้างพื้นที่เริ่มต้น

เปิด tenant เริ่มต้นเพื่อดูเมนูและสิทธิ์ ตรวจฟีเจอร์ที่ยังต้องอัปเกรดก่อนเข้า production

2

ตัดสินใจเรื่อง runtime

Delivery Agent ใช้ส่งของในเกม Server Bot ใช้ sync log แก้ config สำรองข้อมูล หรือควบคุมเซิร์ฟเวอร์

3

เปิดให้ผู้เล่นใช้งาน

เมื่อฝั่งผู้ดูแลพร้อมแล้ว จึงเปิด Player Portal ให้ผู้เล่นเข้าร้านค้า กระเป๋าเงิน และคำสั่งซื้อ

พร้อมเริ่มต้นแล้ว?

สร้างบัญชีฟรี ทดลองดูระบบก่อนตัดสินใจเปิดใช้งานจริง
