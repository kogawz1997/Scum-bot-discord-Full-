# AGENTS.md

## UI Philosophy

- SCUM-inspired, tactical, dark premium
- Clean, readable, not cluttered
- Game-feel but NOT game UI- Must be usable in real production

## Layout Rules

- 8px spacing system
- Balanced spacing everywhere
- No cramped UI- No oversized elements

## Component Rules- All buttons same height system- All cards same radius system- Inputs consistent across app

- Tables readable and structured

## Visual Rules- No neon overload

- No heavy glow
- No messy panels
- Use subtle rugged style only

## UX Rules- Every page must be easy to understand

- Clear primary action
- Dangerous actions must be separated
- Navigation must be obvious

## Security Rules- Do NOT change auth logic

- Do NOT change session logic- Do NOT change API contracts
- Do NOT break TOTP / Discord SSO
- Do NOT remove security UI flows

## Dev Rules- Do NOT rewrite framework

- Do NOT break routes
- Preserve DOM hooks (id, name, data-\*)
- If unsure → wrap instead of replace

## Quality Bar

- Must look like real production- Must not look autogenerated
- Must be clean, balanced, and usable
