Owner + Tenant Access
Secure access

เข้าสู่ระบบผู้ดูแล

หน้าเดียวสำหรับเจ้าของระบบและผู้ดูแลผู้เช่า โดยระบบจะพาไปยังพื้นผิวที่ตรงกับสิทธิ์ของคุณอัตโนมัติหลังเข้าสู่ระบบ

Owner

ดูแลทั้งแพลตฟอร์ม ผู้เช่า การเงิน ความปลอดภัย และเหตุการณ์ระดับระบบ

Tenant Admin

ดูแลเซิร์ฟเวอร์ของตัวเอง คำสั่งซื้อ ผู้เล่น การส่งของ และบอตที่ผูกกับ tenant

Sign in

ล็อกอินเข้าสู่ระบบ

เราใช้ auth, session และ 2FA เดิมของระบบ ไม่มีการลดขั้นตอนด้านความปลอดภัย

ถ้าเปิด 2FA อยู่ ระบบจะขอรหัสยืนยันในขั้นตอนถัดไปโดยอัตโนมัติ
